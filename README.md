# Personal Website

My personal website built with Astro, TypeScript, and CSS.

## Features

- Content Collections for blog management
- Dynamic RSS feed, sitemap, and robots.txt generation
- Light/Dark theme support
- SEO optimized
- Fully responsive design with mobile-first approach

## Running Locally

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:4321](http://localhost:4321) in your browser

## Deployment

This site is deployed on [Vercel](https://vercel.com). Continuous deployment is set up to
automatically build and deploy changes when code is pushed to the main branch.
